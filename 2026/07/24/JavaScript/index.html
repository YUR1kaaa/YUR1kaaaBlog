1、 <script> 标签

在HTML页面嵌入JS脚本需要使用 <script> 标签

2、JavaScript代码执行顺序

浏览器根据文档流 从上到下 逐行解析和显示HTML文档

JavaScript脚本执行顺序也是根据 <script> 标签来确定的

所以js脚本块不必写在html标签内

3、JavaScript基本语法

3.1 标识符

所谓标识符就是名字，JavaScript中标识符包括变量名、函数名、参数名、属性名、类名等

合法标识符（变量命名）规则：

①第一个字符必须是字母、下划线（_）或美元符号（￥）;

②除第一个字符外，其他位置可以使用Unicode字符。

③不能与js关键字、保留字重名

3.2 关键字和保留字

关键字（Keyword）有特定用途

保留字是javascript内部预备使用的一组名字，目前没有具体用途，是为js升级预备用的

3.3 大小写

JavaScript标识符严格区分大小写

4、注释

1 单行注释

2 多行注释

5、定义变量

JavaScript用 var 关键字定义变量

可以一次定义多个变量，用逗号隔开

定义变量后默认初始值 undefined（未定义）

5.1 赋值

也可以在定义变量的同时赋值

5.2 let和const关键字

let 和 const 用于声明变量

let 声明的变量只在其所在代码块中有效，且在同一代码块中不能重复声明（类似局部变量）

const 的功能和 let 相同，但 const 定义的关键字不能修改（相当于定义的是常量）

6、JavaScript数据类型

6.1 数据类型

JavaScript中数据类型可分为两种：

基本数据类型（值类型）：字符串（String）、数字（Number）、布尔（Boolean）、空（Null）、未定义（Undefined）、Symbol；

引用数据类型：对象（Object）、数组（Array）、函数（Function）；

6.2 typeof操作符

使用 typeof 操作符返回变量数据类型

6.3 Symbol类型

ECMAscript6新引入，表示独一无二的值

Symbol类型的值需要使用Symbol（）函数生成

6.4 Object类型

JavaScript中的对象（object）类型是一组由键、值组成的无序集合，定义对象类型需要使用花括号{ }

其中name为键，value为值（类似python字典）

在JavaScript中，对象类型的键都是字符串类型的，值则可以是任意数据类型

可以使用 对象名.键 的形式获取对象中的某个值

6.5 Array类型

数组（Array）是一组按顺序排列的数据组合

数组中的每个值称为元素，数组中可以包含任意类型数据（类似python列表）

在JavaScript中定义数组用方括号[ ]，数组中元素使用逗号分隔

也可以使用 Array（） 函数创建数组

6.6 Function类型

函数（Function）是一段具有特定功能的代码块

函数不会自动运行，需要通过函数名调用
